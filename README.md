# Whack-a-Box Game

A fun click-the-box game built with HTML, CSS, and JavaScript.

## How to Play

1. Click "Start Game" to begin
2. Click on boxes as they appear to score
4. Try to beat your high score!

## Features

- Timer display
- Responsive design

## Installation

1. Clone this repository
2. Open `index.html` in your web browser
3. Start playing!
